# Miles to Kilometers Converter 

A Python GUI application built with Tkinter that converts miles into kilometers. Users can enter a value in miles and instantly see the converted distance in kilometers by clicking a button. 

---

## Technical Skills:

- Python 3
- Graphical User Interface (GUI) using Tkinter
- Input validation to prevent invalid (non-numeric) entries  

---

## Converter Preview:

![Miles to Kilometers Converter Screenshot](converter-preview.jpg)

---

## How The Converter Works:

- Enter the number of miles into the input field.
- Click the Calculate button.
- The equivalent distance in kilometers will be displayed on the screen.

---
  
## How to Run the Program:

- Make sure you have Python 3 installed.
- Clone this repository or download the Python file.
 ```bash
  git clone https://github.com/MichelleRunning/miles-to-km
  cd miles-to-km
  ```
- Run the program using:
  ```bash
  python main.py
  ```
